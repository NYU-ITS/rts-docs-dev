# Research Technology Services @ NYU

## Deploy locally:
On Linux/Macos run:
```
curl -fsSL https://pixi.sh/install.sh | bash
```
or look for alternate methods/platforms [here](https://prefix.dev/docs/pixi/overview#installation).

### Setup environment:
```
pixi shell
```

### Build and serve:
```
npm ci
npm run build
npm run serve
```
